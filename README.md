## Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/car-showcase-app.git
```

### Open Project

```bash
cd car-showcase-app
```

### Install Packages

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```
